# Content
마리오네트 인형을 이용하여 인형극을 만들어 아이들의 교육에 도움이 될 수있도록 하는 프로젝트 
<h3>시나리오 제작툴</h3>
Drag & Drop으로 간단히 동작과, 소리, 배경음악을 정의하여 키오컨트롤러[인형을 조작하는 로봇] 으로 동작데이터와 소리파일을 전송하여 인형을 움직이게 한다.
